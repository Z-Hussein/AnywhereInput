# Remote Mouse Controller
Control your Windows PC's mouse/keyboard from any Android device via web browser. Uses WebSocket & ngrok for cross-network access.

## ✨ Features
*   **Cross-Network Support:** Connect anywhere using ngrok tunneling.
*   **Full Input:** Supports movement, clicks, scrolling, and hotkeys.
*   **Secure:** Token-based authentication.

## ⚙️ Requirements
*   Windows PC with Python 3.9+
*   Android device (modern browser)
*   ngrok account

## 🚀 Quick Start (Windows Users)
1. Run `start_with_ngrok.bat` in PowerShell.
2. The script sets up the environment, generates a token, and starts the server.
3. Copy the displayed public URL to your Android browser and connect!
